Chess Game

A fully interactive chess game that replicates the classic board game with responsive design and standard chess rules. Players can compete in a player-vs-player mode, with the game enforcing check, checkmate, and stalemate logic.

Features

  • Drag-and-drop functionality for moving pieces
  • Visual indicators for valid moves
  • Dynamic turn-based gameplay logic

Technologies Used

  • HTML, CSS, and JavaScript
  • DOM manipulation for piece movements
  • Responsive design using CSS Grid

Challenges Faced

One major challenge was implementing the logic for special chess moves such as castling, en passant, and pawn promotion. This required understanding the intricacies of chess rules and coding them effectively while avoiding bugs.

Outcome

The project successfully demonstrates a functioning chess game with intuitive controls, responsive design, and accurate rule enforcement. It is a great example of combining logic-based problem solving with front-end design principles.
